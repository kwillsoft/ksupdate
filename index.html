My Favorite Development Tools

Though I initially started coding using Python and C# (which I still use), My app dev studio currently primarily consists of the following languages, libraries, & frameworks.

React

React is a JS library used in the development of user interfaces for single-page or mobile applications.

Bootstrap(CSS5)

CSS is a styleshhet language used to describe document presentation. I typically use the Bootstrap CSS framework to enhance it's functionality.

JavaScript(ES6)

Alongside HTML and CSS, JS is one of the three core technologies of the World Wide Web. JavaScript enables interactive web pages and thus is an essential part of web applications

NodeJS

NodeJS is server-side environment that allows me to write scripts outside of the browser using JavaScript.

Express

Express is a Node-based framework I use to write server-side templates for web and mobile applications.

Redux

Redux is a state manager used to extend the capabilities of the React library

PHP

PHP, AKA Hypertext Preprocessor is mainly a server-side scripting language designed for Web development, but also used as a general-purpose programming language

MongoDB

MongoDB is a cross-platform document-oriented database program. I use MONGO as part of my MERN (Mongo-Express-React-Node) setup...

MySQL(MariaDB)

MySQL is a relational database management system (RDMS), I usually utilize in conjunction with PHP, in a LAMP (Linux-Apache-MySQL-PHP) situation.

Unity3D

Unity3D is one of the gaming industry's most widely used game engines-which are frameworks that allow the smooth importing and assembling of assets from various software sources.

BLENDER

Blender is 3D toolset that supports the entirety of the 3D pipeline-modeling, rigging, animation, simulation, rendering, compositing,motion tracking, video editing and game creation.

GIMP/INKSCAPE

GIMP is a raster graphics editor, while Inkscape allows me to manipulate vector images. (GIMP and IS are non-proprietary alternatives to the Adobe PhotoShop/Illustrator stack.)
