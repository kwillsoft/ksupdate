My Favorite Development Tools

Though I initially started coding using Python and Ruby, My app dev studio currently primarily consists of the following languages, libraries, & frameworks.

React

React is a JS library used in the development of user interfaces for single-page or mobile applications.

Sass

Sass is a super-handy CSS preprocessor that provides a lot of flexibility and control aka "syntactic sugar' to the styling process. I dig it.

MJML

MJML helps me to speed with my HTML email template design process. It was designed to aid easing cross-browser compatability.

Webpack

My app development process usuallu won't be complete without using Weback to bundle my dependencies.

Babel

Babel is a necessary tool I used to make sure my JS is 'update-proof. It transpiles the latest JS code into code that can be run on older JavaScript engines.

Redux

Redux is a state manager used to extend the capabilities of the React library. It makes it easier to control the state of multiple components simulateously.

PHP

PHP, AKA Hypertext Preprocessor is mainly a server-side scripting language designed for Web development, but also used as a general-purpose programming language & that language I utilize in the WordPress world.

MongoDB

MongoDB is a cross-platform document-oriented database program. I use MONGO as part of my MERN (Mongo-Express-React-Node) setup...

MySQL(MariaDB)

MySQL is a relational database management system (RDMS), I usually utilize in conjunction with PHP, in a LAMP (Linux-Apache-MySQL-PHP) situation.

Unity3D

Unity3D is one of the gaming industry's most widely used game engines-which are frameworks that allow the smooth importing and assembling of assets from various software sources.

BLENDER

Blender is 3D toolset that supports the entirety of the 3D pipeline-modeling, rigging, animation, simulation, rendering, compositing,motion tracking, video editing and game creation.

ADOBE CC

When it comes to image manipulation, Adobe Photoshop and Illustrator usually work perfectly for me. For video editing, I use After Effects & Sony Vegas (FL Studio 12 & Akai MPC for audio).
